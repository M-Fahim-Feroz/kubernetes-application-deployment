## [1.0.0] - Initial Release
### Added
- Complete Kubernetes application deployment demonstrating declarative manifests.
- Helm chart for parameterized deployments across environments.
- Horizontal Pod Autoscaler (HPA), Ingress, and NetworkPolicy implementations.
- GitHub Actions CI pipeline with Trivy security scanning and Kubeconform manifest validation.
- Makefiles for simplified local deployment and rendering targets.
